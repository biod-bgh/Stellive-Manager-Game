## 프로젝트 개요
AI를 사용해서 게임의 주요 요소들을 테스트 해볼 수 있는 MVP모델 만들기

## 1. 왜 만들었나요?
   프레임워크를 구매해서 만들어보던 버튜버 오토체스 게임의 **피쳐를 테스트**해보려고

## 2. 무슨 특징이 있나요?
   그래픽을 최소화했을 때, '수집요소를 모으고 싶다'는 생각이 드는가?
   '더 가지고 놀고 싶다'라는 생각이 드는가?
   QTE 시스템이 게임에 녹아드는가? 녹아들지 않는다면 어떤 부분이 과하거나 부족한가

 ## 3. 어디까지, 언제까지 개발할 건가요?
    프로그래밍 기본기가 필요해지는 부분이 많아지면 멈출 예정. 기획에 필요한 데이터를 구조화하는 작업은 맘에 들때까지 할 에정.
    코드가 길어져서 여러 파일로 쪼개야 겠다... 라는 생각이 들기 시작하면 추가적인 개발은 하지 않을 생각.
    입력되는 데이터(캐릭터, 적, 특성, QTE 설정 등..)을 쪼개는 작업은 스스로 만족할때까지 수정할 생각.
